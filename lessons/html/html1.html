1. What is HTML?

Welcome to the Byte-Sized Lesson on HTML!

HTML stands for HyperText Markup Language

2. Hypertext?

What is Hypertext? It's textâ€¦ that's hyper! It's like regular text but it can do additional things, like link to other text, include images in the text, and do all of the things that you see on the Web.

3. Markup Language??

And what is a markup language? It's the language that turns regular text into hypertext by marking it up with tags!

For example, here is some regular text: I'm regular text!
Here is regular text that's marked up: <b>I'm regular text!</b>
And it comes out like this: I'm regular text!

4. What is HTML Good For?

HTML is necessary for the internet.

In the beginning, webpages were meant to be static pages that could be shared on the web.

HTML provides all the tools necessary to make these static webpages.

5. Static Webpages? Is that all?

No, static webpages weren't enough!

People wanted webpages with content that was styled and positioned as they pleased!

And that's why today, along with HTML, we also have CSS and Javascript.

6. And it all starts with HTML

But, as useful as they are, CSS and Javascript are nothing without HTML

It's important to have a good grasp of the basics before moving on to more complex things.

And continue on to learn a little bit about the basics.
