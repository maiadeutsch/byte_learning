1. Anatomy of an HTML Web Page

Take a look at this sample HTML page here:

<html>
<head>
</head>
<body>
Hello, World!
</body>
</html>

2. What's going on here?

Copy the code into a simple text document, and save it as myFirstPage.html.

<html>
<head>
</head>
<body>
Hello, World!
</body>
</html>

4. Dissecting the HTML Code

HTML is centered around the idea of tags.

Tags are the things inside of brackets that you see in the code.

Everything in our example is a tag, except for one little line of text: "Hello, World!".

This is the only text that appears in our webpage.

5. More on Tags

Tags remain invisible in the webpage.

That's because tags are just instructions that are meant for the browser, not for the viewer.

Tags generally come in pairs: an opening tag, and a closing tag.

6. Tags EVERYWHERE!

The first tag in a pair is the opening tag, and the second tag is the closing tag.

For example, <body> is an opening tag, and </body> is a closing tag.

"Hello, World!" is inside of the body section of the code.

7. Go and see for yourself!

In fact, everything is inside the HTML section!

That's because every webpage always begins with an opening <html> tag and ends with a closing </html> tag.

Don't believe me? Well hey, you can check this out for yourself!

The easiest way to do this is using Google Chrome.

8. Okay, It Gets Crazy...

Right-click anywhere on a webpage and select 'View Page Source'.

If you're new to HTML, you might be overwhelmed by what you see.

This is because an HTML page often contains inside of it confusing CSS and Javascript.

9. But it Gets Better!

Even pages that may seem simple at first (such as the Google homepage) are actually extremely sophisticated.

Webpages have become quite complex these days, but do not fear!

With patience and perseverence, you'll learn how to navigate through the slush and identify the important things.
