HTML 5 is becoming the new HTML standard and provides many new powerful features!

Swipe to learn more.

Embed Video

One powerful feature is the ability to embed videos within the markup

Example


<video width="320" height="240" controls="controls">
  <source src="movie.mp4" type="video/mp4">>
  <source src="movie.ogg" type="video/ogg">
  <source src="movie.webm" type="video/webm">
</video>

Embed Audio

Another powerful feature is the ability to embed audio within the markup

Example


<audio controls="controls">
  <source src="horse.ogg" type="audio/ogg">
  <source src="horse.mp3" type="audio/mpeg">
</audio>

Draw Graphics

Canvas allows a page to render graphics on the fly

Example


<canvas id="myCanvas" width="200" height="100"
style="border:1px solid #000000;">
</canvas>
